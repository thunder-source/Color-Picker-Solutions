Gradient Provider

CSS gradients let you display smooth transitions between two or more specified colors. It must have atleast two color stops. CSS defines three types of gradients: Linear Gradients (down/up/left/right/diagonally) Radial Gradients (defined by their center) Conic Gradients (rotated around a center point)

Color Picker

Use the color picker by clicking and dragging your cursor inside the picker area to highlight a color on the right. click the swatch to add it to your clipboard. After selecting a color, it will allow to copy the color code as well as provide css code.

Image Color Fetcher

Image color picker allow us to select a color and get the HTML Color Code of this pixel. Under 'Use Your Image'. You can upload your own image paste an image from clipboard, put a picture url in the textbox below.
